# duid_template
duid template


## Install  
```  
cargo install cargo-duid-app  
```  
  
## Create project  
```  
cargo duid-app init --name <PROJECT_NAME>  
```  
  
  ## Develop your project  
```  
cargo duid-app dev // default --host="0.0.0.0" --port=3000
cargo duid-app dev --host <HOST>  --port <PORT>
```
